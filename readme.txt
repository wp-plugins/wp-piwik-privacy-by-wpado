Erstellt den Shortcode [wpado_piwik_privacy] um das Tracking von dem Analyse-Tool Piwik zu deaktivieren.

== Description ==

Es wird vorausgesetzt, dass das Plugin [WP-Piwik](http://wordpress.org/extend/plugins/wp-piwik/) installiert und aktiviert ist. Es wird dem Besucher ermöglicht, das Tracking von dem Analyse-Tool Piwik zu deaktivieren. 
Der Seiteninhaber kann die Deaktivierungsmöglichkeit mit dem Shortcode [wpado_piwik_privacy] einfügen.

Weitere Informationen erhalten Sie unter:
 http://www.wp-agentur-do.de/referenzen-eintrag/wp-piwik-privacy-by-wpado/

== Installation ==

= Installation via Wordpress =

1. Menu "Plugins" -> "Installieren" und nach "WP-Piwik Privacy by WPADO" suchen
2. Installieren klicken

= Manuelle Installation =

1. Plugin herunterladen und entpacken
2. Verzeichnis "wpado-piwik-privacy" nach "/wp-content/plugins/" hochladen.
3. Plugin aktivieren.
